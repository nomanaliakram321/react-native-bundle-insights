# Screenshots Guide

This directory contains screenshots for the README documentation.

## How to Add Screenshots

1. Take screenshots of each feature
2. Save them in this directory with descriptive names
3. Reference them in the main README.md

## Required Screenshots

### Terminal Output
- `terminal-analysis.png` - Running the complete analysis
- `terminal-bundle-treemap.png` - Bundle treemap command output
- `terminal-security.png` - Security scan output
- `terminal-assets.png` - Asset analysis output

### Web Dashboard
- `dashboard-overview.png` - Main dashboard with all tabs
- `dashboard-bundle-treemap.png` - Bundle treemap tab
- `dashboard-dependencies.png` - Dependencies tab
- `dashboard-unused-imports.png` - Unused imports tab
- `dashboard-security.png` - Security tab
- `dashboard-assets.png` - Assets tab

### Features
- `treemap-interactive.png` - Interactive treemap visualization
- `package-breakdown.png` - Package size breakdown
- `optimization-suggestions.png` - Optimization suggestions
- `security-issues.png` - Security issues list
- `asset-optimization.png` - Asset optimization recommendations

## Screenshot Guidelines

- Use high resolution (at least 1920x1080)
- Capture in dark mode for consistency
- Include relevant data (not empty states)
- Crop to show only relevant content
- Use PNG format for better quality
